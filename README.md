# Fusion2 Content (Archive)

[![GitHub](https://img.shields.io/badge/GitHub-Yakuza2635-181717?style=flat&logo=github)](https://github.com/Yakuza2635)
[![License](https://img.shields.io/badge/License-MIT-blue.svg)](LICENSE)

Archive of the previous fusion-content version, plus widget configs and addon settings from the UME config vault.

> **Note:** The active Fusion content repo is [fusion-content](https://github.com/Yakuza2635/fusion-content). This repo preserves the previous version's history and includes integrated UME configs.

---

## What's Included

### Folders (posters, landscapes, squares)

Same content categories as the active repo — Animation Studios, Collections, Genres, Lists, Releases, Streaming Services, Studios — in poster, landscape, and square formats.

### Resource Filters (65 filters)

Media Source, Resolution, Dynamic Range, Audio, Audio Channels, Languages.

### UME Configs (from ume-config-vault)

| Config | Path | Description |
|--------|------|-------------|
| Fusion Widgets | `widgets/fusion-widgets-v0.1.json` | Widget export for Fusion app |
| AIOMetadata | `addons/aiometadata/` | 577 catalogs config |
| AIOStreams | `addons/aiostreams/` | Stream filtering config |
| Streaming Logos | `assets/streaming-logos.json` | Logo reference data |
