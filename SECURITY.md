# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it responsibly.

**Please do NOT open a public GitHub issue for security vulnerabilities.**

Instead, please send an email to the repository owner with:

- A description of the vulnerability
- Steps to reproduce the issue
- Any potential impact

You should receive a response within 72 hours acknowledging receipt of your report.

## Supported Versions

| Version | Supported |
|---------|-----------|
| Latest  | Yes       |

## Security Updates

Security updates will be applied as soon as possible after a vulnerability is confirmed.
